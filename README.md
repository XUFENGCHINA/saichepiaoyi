# 我的作品

本项目由 DeepSeek 制作。

一个集成了计算器、漂移赛车、函数绘图与模组模板的启动导航页，用于快速访问个人开发的多个 Web 工具。

## 项目简介

本项目是一个轻量级 HTML 导航页面，包含启动过渡动画和功能卡片主界面。页面提供四个核心工具入口：

- 计算器：基础数学运算与科学计算
- 漂移赛车：极速驾驶与漂移模拟
- 函数画图器：数学函数图像可视化
- 漂移赛车模组模板：赛车自定义模组开发框架

每个卡片均可独立配置跳转链接，适合作为个人作品集的统一入口。

## 快速开始

1. 下载或克隆本仓库。
2. 编辑 `index.html` 中的 `LINKS` 对象，为每个工具填写对应的网页地址。
3. 将项目部署到任意静态托管服务，或直接在浏览器中打开 `index.html` 预览。

## 自定义说明

- 修改页面标题：搜索 `我的作品` 并替换为自定义名称。
- 调整图标：卡片中的 Emoji 可更换为其他符号或字体图标。
- 新增卡片：复制现有卡片结构，并在 `LINKS` 中添加新键值对。

## 技术实现

- 纯原生 HTML、CSS 与 JavaScript，无第三方库依赖。
- 采用毛玻璃效果、渐变背景与平滑动画。
- 响应式布局，适配桌面与移动设备。
